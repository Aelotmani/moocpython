\subsection{Exercice - niveau

\begin{verbatim}

\end{verbatim}

l'ensemble des objets \texttt{valeur} associés (dans A ou dans B) à un
entier qui soit présent dans (un tuple de) A \emph{et} dans (un tuple
de) B.
